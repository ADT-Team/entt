Sets the value of a given variable.

It must be possible to cast the instance to the parent type of the data member. Otherwise, invoking the setter results in an undefined behavior.
The type of the value is such that a cast or conversion to the type of the variable is possible. Otherwise, invoking the setter does nothing.
